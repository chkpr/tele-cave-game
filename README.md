# Tele-cave-game

An IVR game concept for telemarketers, inspired by text-based adventure games from the 70s. Trapping telemarketers in a cave where they navigate by pressing keys on their keypads. Built to explore Java in telephony while practicing Spring Boot.

## How it works

Launch the game when a telemarketer calls. They get trapped in a cave and must navigate their way out by pressing keys on their keyboard. 

## Stack
- Java 21
- Spring Boot 3
- Maven 
- Twilio
